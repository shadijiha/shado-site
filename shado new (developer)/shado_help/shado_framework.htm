Shado framework :

This is a documentation of all shado_JS_framework.js functions up to this version's release date ()


  1. Getting the root of a number:


    This function allow you to get root of any power of any number. This function takes the number (num) and the power (power) as parameters. The power representes the root intensity (square root, cubic root, ...).

    function root(num, power) {
    return Math.pow(num, 1 / power);
    }

  2. Converting radians to degrees:


    This function allow you to get the degree value of any radian angle. This function takes the initial radian value of the angle (rad) as parameter.

    function radianToDegrees(rad) {
    return (rad * 180) / Math.PI;
    }

  3. Writing something to an HTML element:


    This function alow you to write any text or html code to an html element. This function takes the HTML element id (elementId) and the things to write (argument) as parameters.

    function write(elementId, argument) {
    document.getElementById(elementId).innerHTML = argument;
    }

  4. Putting a number to the power of another number


    This function alow you to put any number to the power of another number. This function takes the base number (base) and the power (exposant) as parameters.

    function exp(base, exposant) {
    return Math.pow(base, exposant);
    }

  5. Getting the square of a number:


    This function alow you to get the square of a number. This function takes the base number (base) as parameter.

    function square(base) {
    return Math.pow(base, 2);
    }

  6. Getting the sin, cos or tan of an angle:


    This function returns the sin, cos or tan of an angle. This function takes the angle value (ang) as parameter. (Note: if you want to use degrees you must include the following statment in your code : angles = "degrees")

    function sin(ang) { // You can replace sin by cos or tan
    if (angles == "degrees") {
          ang = (ang * Math.PI) / 180;
    } else {
         angles = "radians";
    };

    return Math.sin(ang); // This can varry
    }
